# Plan 02 Summary (요약)

**Status:** Completed (완료)
**Objective:** 바닐라 JavaScript를 사용하여 프론트엔드 UI를 백엔드의 `/api/convert` 엔드포인트와 연결합니다.

## 완료된 작업 (Tasks Completed)
- 프론트엔드 기능을 담당하는 `public/app.js`를 생성했습니다.
- 입력된 유튜브 링크를 읽어와 `fetch()`를 사용해 백엔드 API로 전송하는 로직을 작성했습니다.
- 통신 중 버튼 상태 비활성화, 안내 메시지 노출 등 기본적인 UI 피드백 처리를 적용했습니다.
- 응답받은 오디오 데이터를 Blob 형태로 파싱하고 가상의 `<a>` 태그를 생성해 브라우저 단에서 직접 파일 다운로드가 이뤄지도록 연결했습니다.
